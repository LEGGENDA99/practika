# Загрузчик изображений

Приложение для поиска, просмотра и автономного сохранения изображений по ключевым словам. Проект состоит из сервера на Node.js (Express) и веб-интерфейса, сохраняющего контент локально в браузере для работы в оффлайн-режиме.

## Функционал

* Поиск изображений из внешних баз данных по ключевым словам.
* Скачивание бинарных данных картинок напрямую через браузер для обхода ограничений CORS и ошибок 403/404 на стороне сервера.
* Конвертация файлов в формат Base64 и сохранение в `localStorage` для просмотра без доступа к сети.
* Поддержка автодополнения ключевых слов при вводе.
* Обработка нажатия клавиши **Enter** для запуска поиска.
* Автоматическое отслеживание сетевого статуса (Онлайн / Оффлайн).

## Установка и запуск

1. Склонируйте репозиторий или перенесите файлы проекта в рабочую директорию.
2. В корневой папке проекта инициализируйте Node.js приложение и установите необходимые зависимости:

```bash
npm init -y
npm install express 
npm install cors
npm install nodemon --save-dev

```

3. Запустите сервер:

```bash
npm run dev

```

4. Откройте приложение в браузере по адресу:
`http://localhost:3000`

## Использование

1. Введите в поле поиска одно из доступных ключевых слов (`dogs`, `cats`, `nature`, `anime`).
2. Нажмите кнопку **Найти** или клавишу **Enter**.
3. В появившемся списке выберите нужную картинку по названию и нажмите **Скачать** (или кликните по названию).
4. Загруженная картинка отобразится в блоке просмотра и автоматически добавится в раздел «Сохраненные картинки».
5. При отключении интернета приложение перейдет в оффлайн-режим, позволяя просматривать и удалять ранее сохраненные файлы.
